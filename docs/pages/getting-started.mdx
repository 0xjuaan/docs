## Installation

Install in an existing [foundry](https://getfoundry.sh/) project:

```bash
forge install hyperliquid-dev/hyper-evm-lib
echo "@hyper-evm-lib=lib/hyper-evm-lib" >> remappings.txt
```

## Quickstart
