## Overview

<img src="https://i.ibb.co/Q7X6CK8x/image.png" alt="image" style={{ width: '60%', height: 'auto' }} />

`hyper-evm-lib` is the all-in-one developer library for building smart contracts on HyperEVM.

### Key Features
- Libraries to simplify the process of writing EVM contracts which interact with HyperCore
    - See [CoreWriterLib](/dev/corewriter) and [PrecompileLib](/dev/precompiles)
- A robust testing framework to simulate HyperCore interactions in foundry tests 
    - See [Testing Framework](/testing/overview)
- A trustless registry for resolving token and spot market IDs from EVM addresses
    - See [TokenRegistry](/dev/token-registry)
